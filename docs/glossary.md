# Glossary

Terms used consistently across this repository.

| Term | Definition |
|---|---|
| **ORB** | Opening Range Breakout — the high/low established during the first N minutes of the RTH session. |
| **ORBH / ORBL** | Opening Range High / Low for a given timeframe (15m, 30m, 60m). |
| **15m ORB** | High & low of the first 15 minutes of RTH (09:30–09:45 EST). |
| **30m ORB** | High & low of the first 30 minutes of RTH (09:30–10:00 EST). |
| **60m ORB** | High & low of the first 60 minutes of RTH (09:30–10:30 EST); typically equal to the **IB**. |
| **IB (Initial Balance)** | The 09:30–10:30 EST high/low range. IBH = high, IBL = low. |
| **IB Range** | IBH minus IBL in points. |
| **Contained** | Price stays inside IB the rest of the session. |
| **Extended** | Price breaks out of IB (up or down) and accepts outside. |
| **Failed IBH Break** | Price breaks IBH but is rejected back inside IB. |
| **Failed IBL Break** | Price breaks IBL but is rejected back inside IB. |
| **VWAP** | Volume-Weighted Average Price; intraday session VWAP unless noted. |
| **ON High / ON Low** | Overnight session high/low (18:00 EST prior day → 09:30 EST). |
| **Wick Shelf** | A price level marked by a clear single-candle wick rejection. |
| **Pivot Shelf** | A level that has acted as both support and resistance intraday. |
| **Proven Level** | A level tested and held during the session. |
| **Broken Level** | A level lost in-session that subsequently flipped polarity. |
| **Fragile Level** | A level currently being tested with no confirmation yet. |
| **Acceptance** | Price trades and holds beyond a level for a sustained period. |
| **Rejection** | Price tags a level and reverses within 1–3 bars. |
| **Two-Step Reversal** | Failed IBH/IBL + lost VWAP = high-confidence reversal regime. |
| **Tape** | Cross-asset confirmation: ES / NQ / RTY / VIX / TLT / DXY / $TICK. |
