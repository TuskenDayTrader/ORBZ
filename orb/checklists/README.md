## Planned Pre-Market Checklist Sections

The pre-market checklist is expected to include:

1. **Market context check** — futures direction, overnight range, news events
2. **Key level identification** — prior day high/low, VWAP, ORB zone, IB
3. **Session overlap awareness** — any active London or Asia influence
4. **Trade plan** — anticipated setup, direction, entry trigger, stop, target
5. **Risk check** — daily loss limit review, position size pre-calculation
6. **Mental/readiness check** — (accessibility feature — brief self-assessment)

## Planned Trade Entry Checklist Sections

A quick go/no-go list to run through immediately before entering a trade:

- [ ] Opening range fully formed?
- [ ] Breakout direction confirmed?
- [ ] Volume confirmation checked?
- [ ] False breakout filters checked?
- [ ] Stop loss level identified?
- [ ] Position size calculated?
- [ ] Daily loss limit still available?
- [ ] Higher-timeframe context aligned?
