## How to Use This Template

1. Copy this file.
2. Rename it: `YYYY-MM-DD-trade-N.md` (e.g., `2026-05-16-trade-1.md`).
3. Fill in every field. Write `—` for anything that doesn't apply.
4. Save in your session folder or personal journal folder.
5. Never leave a field blank — use `—` or `unknown` so the record stays clean.
