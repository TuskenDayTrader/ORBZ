## Fast glossary

| Term | Simple meaning |
|---|---|
| ORB | Opening Range Breakout |
| Breakout | Price moves outside the opening range |
| Retest | Price comes back to test the broken level |
| Stop loss | Your exit if the trade is wrong |
| Target | Your planned profit level |
| Risk-Reward (R:R) | Risk compared to possible reward |
