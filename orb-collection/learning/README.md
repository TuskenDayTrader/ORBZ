## Export workflow

From repository root:

```bash
./orb-collection/learning/export-pdf.sh
```

This script renders `course-manuscript.md` to `dist/orb-course-v1.pdf` using `pandoc`.

## Who this is for

- New traders
- Traders who need simple language
- Traders who learn best with short steps

## How to use this course

- Read one file at a time.
- Keep your charts simple.
- Use paper trading first.
- Do not move to the next step until the current step feels easy.
