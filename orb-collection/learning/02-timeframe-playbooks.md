# 02) Timeframe Playbooks (5m, 15m, 30m, 60m)

## Quick comparison

| Timeframe | Opening range window | Typical use | Notes |
|---|---|---|---|
| 5m | 09:30-09:35 | Fast entries | More noise, faster decisions |
| 15m | 09:30-09:45 | Beginner-friendly base model | Balanced speed and quality |
| 30m | 09:30-10:00 | Higher confirmation | Fewer but often cleaner setups |
| 60m | 09:30-10:30 | IB-style context trades | Slowest, most selective |

## Learning path by timeframe

Start simple. Add speed later.

1. Learn 15m first for 2+ weeks.
2. Add 30m to improve patience and filtering.
3. Add 5m only after you can follow rules without emotion.
4. Add 60m for context and selective high-quality setups.

## Playbook format (same for every timeframe)

1. Mark OR high and OR low when the window closes.
2. Wait for a breakout close.
3. Check breakout volume.
4. Enter on close or on retest.
5. Place stop loss at the opposite side of setup structure.
6. Take profit using fixed R targets (1R, 2R, etc.).

## 5m playbook

- Good for active traders.
- Require strict stops.
- Skip weak low-volume breakouts.

## 15m playbook

- Best starter playbook.
- Fewer fake moves than 5m.
- Use this as your default training model.

## 30m playbook

- Good for clean structure.
- Helps reduce overtrading.
- Works well with breakout + retest entries.

## 60m playbook

- Use for strong session context.
- Best when trend day develops.
- Requires patience; not many signals.
